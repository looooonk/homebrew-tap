# Looooonk Tap

## How do I install these formulae?

`brew install looooonk/tap/<formula>`

Or `brew tap looooonk/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "looooonk/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
